# KametiBook

KametiBook is a Flutter mobile app for managing local committee / VC / kameti systems.

## Features

- Splash and onboarding flow
- Mock authentication UI
- Bottom navigation shell
- Home dashboard
- My Kametis list
- Create Kameti form with validation and pool calculation
- Kameti details screen with placeholders for future modules
- Profile and notifications placeholders

## Run

Install Flutter, then run:

```sh
flutter pub get
flutter run
```

If platform folders are missing in a fresh checkout, run:

```sh
flutter create .
flutter pub get
flutter run
```
